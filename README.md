# REST API for generating stories using AWS Bedrock

This repo contains an example REST API with one endpoint which can generate
stories using AWS Bedrock in response to a POST request containing a prompt.

## AWS Services Used

- Bedrock
- API Gateway
- Lambda
- CDK
